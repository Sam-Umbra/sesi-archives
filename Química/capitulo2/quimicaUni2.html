Balanceamento Químico

Em diversas situações na química quando lidamos com reações químicas, a equação da reação está incompleta ou não balenceada, sendo necessário que seja balanceada para que a reação possa acontecer de forma correta. Para isso nós temos que transformar a equação em sua menor forma possível, na qual a reação ainda ocorra, utilizando a menor quantidade de mols possível em ambos os lados da reação, Reagentes → Produtos.

Em todo o processo da reação, em um sistema fechado, a mesma quantidade de elementos químicos de ver permanecer a mesma antes e depois da reação, nos casos no qual não há está igualdade nos balanceamos a equação. Exemplo:


CH4(g) + O2(g) → CO2(g) + H2O(L)

Aqui é possível perceber que no lado dos reagentes existem apenas 4 Hidrogênios, 1 Carbono e 2 Oxigênios, enquanto no lado dos produtos existem 2 Hidrogênios, 1 Carbono e 3 Oxigênios, sendo possível perceber que não existe uma balança.
Então o que deve ser feito é acrestar mols na equação, os números grandes, lembrando de que os números pequenos não devem ser alterados em hipotese alguma pois representam a substância em si e caso sejam alterados a substância também será mudada.

CH4(g) + 2 O2(g) → CO2(g) + 2 H2O(L)

Agora sim a equação está balanceada e a reação poderá acontecer, pois como é possível ver caso multipliquemos os números pequenos por suas respectivas quantidades de mols haverá uma balança. Dessa forma alcançamos a equação padrão da reação e caso seja necessário aumentar a quantidade de substâncias na reação, como por exemplo a dobrar, será necessário multiplicar os mols de toda equação.

2 CH4(g) + 4 O2(g) → 2 CO2(g) + 4 H2O(L)

Esquiometria

É a forma de calcular/quantificar as substâncias usadas e geradas em uma reação química, e como a massa se transforma dentro da reação de um reagente para um produto. Dentro de uma equação química é comum utilizar mols para medir quantas partículas/átomos/moleculas existem dentro da reação, visto que 1 mol é equivalente a 6x1023 destas pequenas coisas que compoem a matéria.

Cada mol de diferentes substâncias possui uma massa diferente e é para isso que utilizamos a Esquiometria, contudo o calculo em si é "extenso" então vamos pular certas etapas. Resumidamente se usa a Massa Molar, aproximadamente o mesmo valor da Massa Atômica na tabela periódica, de cada elemento que compoe uma substância e então é feito a soma das massas de cada átomo/elemento, depois é multiplicado pela quantidade de mols respectiva da subtância na equação.

O próximo passo para saber a proporção da matéria é fazer uma regra de três, se 3 mols de X viram 1 mol de Y então a massa de X vai virar, proporcionalmente, a massa de Y. Exemplo: Segundo a equação abaixo quantos gramas de N2(g) serão gerados caso seja utilizado 100g de N2H4(L).


CH4(g) + 2 O2(g) → CO2(g) + 2 H2O(L)

Massa Molar de CH4(g)=
C: 12g x 1 átomo = 12g
H: 1g x 4 átomos = 4g
Soma = 16g.

Massa Molar de CO2(g)=
C: 12g x 1 átomo = 12g
O: 16g x 2 átomos = 32g
Soma = 44g.

Regra de 3:
1 CH4(g) - CO2(g)
100g - X

16g - 44g
100g - X

16X = 44x100
16X = 4.400
X = 4.400/16 X = 275g de CO2(g) gerados.

Ligações Covalente

Um dos 3 tipos de ligação química, no qual há o compartilhamento de eletrons entre dois ou mais elementos químicos para alcançar a Regra do Octeto, 8 eletrons na Camada de Valência, e formar substâncias. Ela só pode ocorrer entre os seguintes elementos:

  • Ametal e Ametal (C, N, P, O, S, F, Cl, Br e I).
  • Ametal com hidrogênio.
  • Hidrogênio com hidrogênio.

Sendo representada em 3 fórmula:

  • Fórmula Molecular: H2.
  • Fórmula Eletrônica: H''H.
  • Fórmula Estrutural: H-H.

Todas essas fórmulas partem de uma ligação que ocorre na Camada de Valência.

Camada de Valência

A Camada de Valência consiste-se nos elétrons encontrados nos sub-níveis (Divisões dentro das camadas da eletrosfera dos átomos no Modelo de Bohr) mais energeticos, segundo o Diagrama de Linus Pauling, são com eles que as ligações vão ocorrer. Exemplo:

Ligação de Cl com Cl (Cl2), ambos possuem 7 elétrons na camada de valência então os dois vão compartilhar 1 de seus elétrons para alcançar o octeto.
Cl-Cl

Entalpia

Definição

É a quantidade de calor absorvido ou liberado dentro de uma reação química ou física, sendo representado pela variação das entalpias iniciais/dos reagentes e as finais/dos produtos. Sendo medido em Joule(J) ou Caloria (Cal).

Fórmula Padrão e Classificações

ΔH = Hp - Hr

Sendo classificada em:

  • Exotérmico, processo que libera calor: ΔH < 0.
  • Endotérmico, processo que absorve calor: ΔH > 0.

Estado Padrão

A forma mais estável de um elemento químico ou substância encontrada na natureza em uma pressão de 1 bar e 298,15o. Exemplos:

  • Alúminiosólido;
  • Gás Oxigênio (O2).
  • Carbono (Grafite).

Entalpia Padrão de Formação

É a variação de entalpia quando um mol de uma substância é formado a partir de seus elementos no Estado Padrão. Fórmula: ΔHof = ∑HoFp - ∑HoFr

Dentro de uma equação química é utilizado uma tabela com os valores de entalpia já calculados das substâncias para calcular a variação, caso não seja disponibilizado algum dos valores é porque ele é 0. Exemplo:

Fe2O3(s) + 3 CO(g) → 2 Fe(s) + 3 CO2(g)
-824,2 Kj/Mol + -110,5Kj/Mol → 0 Kj/Mol + -393,5 Kj/Mol

ΔHof = [2 x (0) + 3 x (-393,5)] - [-824,2 + 3 x (-110,5)]
ΔHof = -1.180,5 - [-1.155,7]
ΔHof = -24,8 Kj/Mol

Combustão - Energia de Ligação

É o calculo da energia necessária que deve ser absorvida para romper as ligaçãoes dos reagentes (processo endotérmico) e da energina que deve ser liberada para formar novas ligações (processo exotermico) dentro de uma reação química.

  • Ligações Rompidas (Reagentes): ΔH > 0.
  • Ligações Formadas (Produtos): ΔH < 0
  • Fórmula: ΔH = ∑Hp - ∑Hr

Para fazer o calculo é necessário identificar todas as ligações presentes usando das fórmulas estruturais (lembram-se? Caso não olhem novamente o conteúdo de ligações covalentes), apartir disso é necessário fazer os somatórios utilizando os valores de entalpia de uma outra tabela referente a energia de ligação, considerando os valores como negativos no somatório das ligações formadas. Exemplo:

CH4(g) + 2 O2(g) → CO2(g) + 2 H2O(L)
ΔH = [2 ligações x (-743 kJ) + 4 ligações x (-463 kJ)] + [4 ligações x (412 kJ) + 2 ligações x (496 kJ)]
ΔH = -3.338 + 2640
ΔH = -698 kJ
